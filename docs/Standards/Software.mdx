The following are some software stackups that have been used by other projects in the past. 

* **[Platformio](https://platformio.org/)** - A popular IDE for embedded development that supports multiple platforms including Arduino.
* **[VSCode](https://code.visualstudio.com/)** - A lightweight but powerful source code editor with support for many programming languages and extensions.
* **[MkDocs Material](https://squidfunk.github.io/mkdocs-material/)** - Easy to manage documentation generator to make documentation websites.
* **[GitHub](https://github.com/)** - A web-based platform used for version control and collaboration.
* **[GitHub Desktop](https://desktop.github.com/download/)** - A git client that can be used for version control and collaboration.
* **[OnShape](https://www.onshape.com/)** - A cloud-based CAD platform for mechanical design.
* **[KiCad](https://www.kicad.org/)** - Open source PCB design software for making custom PCBs.
* **[ClickUp](https://clickup.com/)** - A project management tool.
